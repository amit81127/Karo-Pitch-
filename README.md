# Karo Pitch 🚀

Karo Pitch is a dynamic and empowering platform designed to bridge the gap between early-stage startup founders—especially from Tier-2 and Tier-3 cities in India—and top-tier investors. It serves as a structured ecosystem where startups can apply, get shortlisted, and pitch directly to investors, opening doors to vital funding and scaling opportunities.

This project is a modern web application built with **Next.js**, showcasing a clean, "media-style" UI, smooth animations, and a responsive design tailored to engage both founders and investors.

## 🌟 Key Features

-   **Seamless Application Flow:** Clear step-by-step guidance for founders to submit their pitch decks and startup details.
-   **Engaging Animations:** Powered by `framer-motion`, the platform features dynamic backgrounds, scroll-based reveal animations, and interactive hover effects.
-   **Infinite Logo Marquee:** A smooth, continuous scroll showcasing prominent investor networks and venture capital firms.
-   **Modern, Clean UI:** Designed with a striking red-centric theme (`#E11D2E`), utilizing modern typography, glassmorphism hints, and a layout optimized for clarity and trust.
-   **Responsive Layout:** Fully optimized for mobile, tablet, and desktop viewing experiences.

## 🛠️ Tech Stack

-   **Framework:** [Next.js (App Router)](https://nextjs.org/) (v16+)
-   **Library:** [React 19](https://react.dev/)
-   **Styling:** [Tailwind CSS v4](https://tailwindcss.com/)
-   **Animations:** [Framer Motion](https://www.framer.com/motion/)
-   **Icons:** [Lucide React](https://lucide.dev/)
-   **Language:** [TypeScript](https://www.typescriptlang.org/)

## 🚀 Getting Started

To get a local copy up and running, follow these simple steps:

### Prerequisites

Make sure you have Node.js installed on your machine. We recommend using `npm`, `yarn`, `pnpm`, or `bun`.

### Installation

1.  Clone the repository or download the source code.
2.  Navigate to the project directory:
    ```bash
    cd karo-pitch
    ```
3.  Install the dependencies:
    ```bash
    npm install
    # or
    yarn install
    # or
    pnpm install
    # or
    bun install
    ```

### Running the Development Server

Start the application in development mode:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result. The page will automatically re-render when you make edits to the components located in `app/` and `components/`.

## 📂 Project Structure

-   `app/`: Contains the Next.js App Router entry points (e.g., `layout.tsx`, `page.tsx`).
-   `components/`: Houses all the reusable UI components like `Hero`, `About`, `HowItWorks`, `Investors`, `CTA`, and more.
-   `public/`: Contains static assets like images and fonts.

## 🤝 How It Works

1.  **Apply with your pitch deck:** Submit your startup details and presentation through the platform.
2.  **Get shortlisted by KaroStartup:** Our team rigorously reviews applications to select the most promising ventures.
3.  **Pitch live to investors:** Shortlisted founders earn the opportunity to present directly to a network of active investors.
4.  **Raise funding and scale:** Connect, secure capital, and grow your business.
